My Projects
🧠
Final Year Research Project · Full-Stack · AI / Emotion Recognition
EMEXA — Emotion-Aware MCQ Assessment System Final Year
✦ What it does
  • Real-time facial emotion recognition while students take quizzes — adapts Socratic hint delivery based on detected affect.
  • Teacher & Student roles: teachers generate AI-powered MCQs via Gemini, students get personalised learning paths.
  • Gamification, wellness activities, PDF report generation, and real-time notifications via Socket.io.
  • K-Means clustering for distractor analysis and ML-based final mark prediction.
✦ Technical Highlights
  • Frontend: React 19 + Vite + Tailwind CSS + Recharts + Socket.io Client + jsPDF
  • Backend: Node.js + Express.js + Socket.io + Nodemailer + Multer + Cloudinary
  • Database: MongoDB + Mongoose (20+ models: emotion logs, quiz attempts, learning paths…)
  • AI/ML: Google Gemini · Hugging Face · Groq LLaMA 3.3 70B · Brevo Email
